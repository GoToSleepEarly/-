并查集的思路：
1、一般想到联通关系的就是并查集，即通过某些关联，将不同组合并，我们只关心连通性，不关心连同的路径。
2、需要弄清连通的对象，一般是数组的下标，也可以用Map表示具体的值。
3、遇到连通的反向——“分解”，可以考虑逆向思维，从后往前推导。

如果需要连通的过程或者顺序，改用搜索算法（BFS DFS 拓扑排序）
